# ShoppingMall


■ 프로젝트 제목
  - html, css, javascript로 쇼핑몰 홈페이지 제작

■ 프로젝트 설명
  - 반응형 홈페이지 X
     
  - 로그인, 로그아웃 기능.
  - 회원 가입 기능.
    ( 아이디: 6자리 이상,
      비밀번호: 숫자 8자리 이상 )
  - 장바구니 기능.
  - 구매하기 기능.
  - 더보기 버튼 기능.
  - 주문 페이지 (배송정보, 쿠폰/마일리지, 결제방법, 주문상품 정보로 구성)
  - 주문 페이지 유효성 체크.
  - 다음(daum) 우편번호 api 활용하여 배송지 주소 검색.

■ 사용 방법
  - 오른쪽 하단 Environments > github-pages 클릭 > view deployment 클릭
